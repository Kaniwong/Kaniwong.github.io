hexo next主题链接带有%20

发表于 2020-03-28 | 分类于 hexo

问题

使用上next主题以后，在本地运行起来发现链接后面都带有一个 %20 ，导致了跳转过后都直接404了。

原因

默认的主题并没有这个问题，所以想必是next主题自带的bug，所以打开 /themes/next/_config.yml，直接Ctrl+F 查找 menu ，通过比对两个主题的区别可以发现next主题每一个链接后面都有空格，所以猜想是这个原因导致了，修改过后果然就行了。

解决方法

去除掉链接后面的空格即可

# hexo # next # 主题
hexo 报错 FATAL can not read a block mapping entry; a multiline key may not be an implicit key at line 103, column 16
